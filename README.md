# 项目简介

## 核心功能
- 用户端功能：
  - 首页搜索与登录
  - 顶部菜单栏包含网站首页、公益产品（含公益基金、乡村振兴）、公益新闻、我们的伙伴、联系我们（建议反馈）
  - 卡片形式展示公益宣传视频与最新消息
- 管理员端功能：
  - 管理员/角色/用户的权限与账号管控
  - 友情链接管理
  - 公益产品信息维护
  - 爱心基金（类型+信息）管理
  - 新闻信息维护
  - 乡村振兴（类型+信息）管理
  - 公益新闻类型管理
  - 合作伙伴信息维护

## 适用场景
- 乡村振兴项目信息发布与管理
- 公益产品展示与推广
- 公益新闻传播
- 爱心基金管理
- 公益合作伙伴展示
- 用户反馈收集

## 技术特点
- 基于SpringBoot + Vue技术栈构建
- 双端分离架构设计
- 完整的权限管理体系
- 模块化功能设计
